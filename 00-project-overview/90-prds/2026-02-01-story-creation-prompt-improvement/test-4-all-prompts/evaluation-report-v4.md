## Evaluation Framework

### Metrics Used:
1. **TQS (Text Quality Score)** - 0-10
   - Word count appropriateness for age
   - Narrative coherence and arc
   - Onomatopoeia presence (mandatory for ages 0-4)
   - Participation elements (mandatory for ages 0-4)
   - Sensory vocabulary count
   - Coping strategy inclusion (for challenging themes)

2. **IPS (Image Prompt Score)** - 0-10
   - Detail level (40-100 words)
   - Character consistency
   - No visual text/letters
   - Atmosphere matching story context
   - Age-appropriate complexity

3. **SMS (Subjective Measure Score)** - 0-10
   - Emotional engagement
   - Relatability
   - Overall "spark" and fun
   - Logical grounding (no forced elements)

4. **AUS (Alignment with User Intent Score)** - 0-10 (NEW)
   - Magic used appropriately (requested vs. forced)
   - Helper character matches story world
   - Respects realistic vs. fantasy context
