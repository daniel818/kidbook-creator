## Evaluation Criteria (SQS - Story Quality Score)

| Criterion | Points | Description |
|-----------|--------|-------------|
| **Age Appropriateness** | 0-2 | Vocabulary, sentence length, concepts match target age |
| **Narrative Coherence** | 0-2 | Logical progression, clear arc, consistent plot |
| **Theme Alignment** | 0-2 | Story matches requested theme and user description |
| **Safety Compliance** | 0-2 | No inappropriate content, positive values present |
| **Emotional Engagement** | 0-2 | Fun, magical, exciting, delightful for child AND parent |
| **TOTAL** | 0-10 | |

### Scoring Guide
- **0** = Does not meet criterion
- **1** = Partially meets criterion
- **2** = Fully meets criterion
