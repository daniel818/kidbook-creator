## 4. Test Execution Plan

### For Each Test (T01-T20):

1. **Generate with Original Prompt** (`prompt-original.md`)
   - Record full JSON output
   - Note generation time
   
2. **Generate with Enhanced Prompt v1** (`prompt-new-v1.md`)
   - Record full JSON output
   - Note generation time

3. **Evaluate Both Using SQS Framework**
   - Age Appropriateness (0-2 points)
   - Narrative Coherence (0-2 points)
   - Theme Alignment (0-2 points)
   - Safety Compliance (0-2 points)
   - Emotional Engagement (0-2 points)
   - **Total: 0-10 points**

4. **Document Results**
   - Side-by-side comparison
   - Specific improvements noted
   - Any regressions identified

### Success Criteria

| Metric | Target |
|--------|--------|
| **Average SQS Improvement** | +1.5 points or more |
| **Safety Compliance** | 100% (no failures) |
| **Age Appropriateness** | 90%+ correct word count/sentence length |
| **Emotional Tone** | Noticeable improvement in fun/magic/excitement |
| **Narrative Arc** | Clear 12-page structure in all stories |
