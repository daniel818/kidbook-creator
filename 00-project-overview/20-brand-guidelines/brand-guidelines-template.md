# KidBook Creator Brand Guidelines

**Company**: KidBook Creator  
**Tagline**: "Wo jedes Kind der Held seiner eigenen Geschichte wird" (DE) / "Where every child becomes the hero of their own story" (EN) / "כאן כל ילד הופך לגיבור של הסיפור שלו" (HE)  
**Positioning**: Magical. Personal. Safe.  
**Target Market**: Parents, grandparents, families (Germany, Austria, Israel)  
**Markets**: Germany (primary), Austria, Israel (Hebrew)

---

## Logo

### Variants
- **Full logo**: "Sparkee Studios" wordmark with icon
- **Icon mark**: Standalone icon for app/favicon usage
- **Monochrome**: Black/white versions for single-color applications

### Clear Space & Minimum Sizes
- **Clear space**: Minimum 1x the logo height around all sides
- **Minimum sizes**: 
  - Web: 120px width (full), 32px height (icon)
  - Print: 25mm width (full), 8mm height (icon)
  - Favicon: 32x32px, 64x64px, 180x180px

### Usage Guidelines
- **Preferred backgrounds**: White, Gray 50, Gray 900, or Primary Blue
- **Don'ts**: 
  - No stretching or distorting proportions
  - No effects (drop shadows, gradients, outlines)
  - No color changes outside approved palette
  - No placement on busy photographs without overlay
  - No rotation or perspective distortion

## Color Palette

### Primary Colors
- **Magical Purple**: `#8B5CF6` (Violet 500)
  - Usage: Primary brand color, CTAs, magical elements, hero sections
  - Contrast: 4.5:1 on white backgrounds (WCAG AA compliant)
  - Hover state: `#7C3AED` (Violet 600)
  - Active state: `#6D28D9` (Violet 700)
  - Personality: Magical, creative, imaginative

- **Warm Orange**: `#F59E0B` (Amber 500)
  - Usage: Accents, highlights, playful elements, secondary CTAs
  - Contrast: 4.5:1 on white backgrounds
  - Hover state: `#D97706` (Amber 600)
  - Personality: Warm, friendly, energetic

- **Sky Blue**: `#3B82F6` (Blue 500)
  - Usage: Trust elements, informational content, links
  - Contrast: 4.5:1 on white backgrounds
  - Hover state: `#2563EB` (Blue 600)
  - Personality: Trustworthy, calm, reliable

### Neutral Scale
- **Gray 900**: `#111827` - Headings, primary text
- **Gray 700**: `#374151` - Body text
- **Gray 500**: `#6B7280` - Secondary text, placeholders
- **Gray 300**: `#D1D5DB` - Borders, dividers
- **Gray 100**: `#F3F4F6` - Backgrounds, cards
- **Gray 50**: `#F9FAFB` - Page backgrounds
- **White**: `#FFFFFF` - Primary backgrounds

### Semantic Colors
- **Success**: `#10B981` (Green 500) - Success messages, book completed
- **Warning**: `#F59E0B` (Amber 500) - Warnings, attention items
- **Error**: `#EF4444` (Red 500) - Errors, validation issues
- **Info**: `#3B82F6` (Blue 500) - Tips, helpful information

### Illustration Palette (Story Books)
- **Soft Pastels**: For children's book illustrations
  - Pastel Pink: `#FCA5A5`
  - Pastel Blue: `#93C5FD`
  - Pastel Green: `#86EFAC`
  - Pastel Yellow: `#FDE047`
- **Usage**: AI-generated illustrations, book previews, decorative elements

### UI State Colors
- **Default**: Primary/Secondary as defined
- **Hover**: Darker shade (-100 in Tailwind scale)
- **Active**: Darkest shade (-200 in Tailwind scale)
- **Disabled**: Gray 300 background, Gray 400 text
- **Focus**: Primary Blue with 2px offset ring, 3:1 contrast

### Contrast Ratios (WCAG 2.1 AA)
- **Normal text** (16px+): 4.5:1 minimum
- **Large text** (24px+): 3:1 minimum
- **UI components**: 3:1 minimum
- **Focus indicators**: 3:1 minimum vs adjacent colors

## Typography

### Font Family
- **Primary (UI)**: Inter (sans-serif)
  - Usage: Website UI, buttons, navigation, forms
  - Fallback: system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif

- **Display (Headlines)**: Poppins (sans-serif)
  - Usage: Hero headlines, section titles, marketing materials
  - Weights: 600 (Semibold), 700 (Bold)
  - Personality: Friendly, modern, approachable

- **Story Text (Books)**: Georgia or Merriweather (serif)
  - Usage: Inside storybooks for readability
  - Personality: Classic, readable, timeless

### Font Weights
- **Bold**: 700 - H1, H2, strong emphasis
- **Semibold**: 600 - H3, H4, buttons, labels
- **Medium**: 500 - H5, H6, navigation
- **Regular**: 400 - Body text, paragraphs

### Type Scale

#### Desktop (≥1024px)
- **H1**: 48px / 56px line-height, Bold (700)
- **H2**: 36px / 44px line-height, Bold (700)
- **H3**: 30px / 38px line-height, Semibold (600)
- **H4**: 24px / 32px line-height, Semibold (600)
- **H5**: 20px / 28px line-height, Medium (500)
- **H6**: 18px / 26px line-height, Medium (500)
- **Body**: 16px / 24px line-height, Regular (400)
- **Small**: 14px / 20px line-height, Regular (400)
- **Tiny**: 12px / 16px line-height, Regular (400)

#### Mobile (<640px)
- **H1**: 36px / 44px line-height
- **H2**: 30px / 38px line-height
- **H3**: 24px / 32px line-height
- **H4**: 20px / 28px line-height
- **Body**: 16px / 24px line-height
- **Small**: 14px / 20px line-height

### Letter Spacing
- **Headings**: -0.02em (tighter)
- **Body**: 0 (normal)
- **Uppercase labels**: 0.05em (wider)

## Imagery & Iconography

### Photography Style
- **Subject matter**: Parents and children reading together, family moments, book unboxing, children's reactions
- **Tone**: Warm, joyful, authentic, magical moments
- **Composition**: Natural, candid, emotional connections
- **People**: Real families, diverse ages/ethnicities/backgrounds, genuine expressions of joy and wonder
- **Lighting**: Soft, warm, natural light (golden hour preferred)
- **Settings**: Home environments, cozy reading nooks, bedtime scenes
- **Avoid**: Overly staged, stock photo clichés, sterile environments, forced smiles

### Illustration Style
- **Style**: Whimsical, child-friendly, storybook-inspired
- **Usage**: Hero sections, feature explanations, empty states, book samples
- **Color**: Soft pastels with magical purple and warm orange accents
- **Tone**: Playful, imaginative, warm, safe
- **Characteristics**: 
  - Rounded shapes (no sharp edges)
  - Soft shadows and gradients
  - Magical elements (stars, sparkles, clouds)
  - Diverse characters and families
  - Age-appropriate and child-safe

### Icon System
- **Library**: Lucide Icons (consistent stroke-based design)
- **Sizes**: 16px, 20px, 24px, 32px, 48px
- **Stroke width**: 2px (consistent across all sizes)
- **Style**: Outline/stroke (not filled)
- **Color**: Inherit from parent or Gray 700 default
- **Usage**: Navigation, features, UI elements, social media

### Book Mockups
- **Formats**: Physical hardcover books, digital PDF previews, 3D book mockups
- **Context**: Real-world usage (bedtime reading, gift giving, family moments)
- **Quality**: High-resolution, realistic textures (paper, binding)
- **Presentation**: 
  - Open book spreads (show 2 pages)
  - 3D angled views with soft shadows
  - In-context (child holding book, on bedside table)
  - Gift packaging (wrapped with ribbon)

## Layout & Spacing

### Grid System
- **Container max-width**: 1280px (max-w-7xl)
- **Padding**: 
  - Mobile: 16px (px-4)
  - Tablet: 24px (sm:px-6)
  - Desktop: 32px (lg:px-8)

### Spacing Scale (4pt base)
- **space-1**: 4px
- **space-2**: 8px
- **space-4**: 16px
- **space-8**: 32px
- **space-12**: 48px
- **space-16**: 64px
- **space-24**: 96px

### Section Spacing
- **Between sections**: 64px mobile, 96px desktop (py-16 md:py-24)
- **Within sections**: 32px mobile, 48px desktop
- **Component spacing**: 16px-24px

### Border Radius
- **Small**: 6px (rounded-md) - Inputs, small buttons
- **Medium**: 8px (rounded-lg) - Cards, large buttons
- **Large**: 12px (rounded-xl) - Hero cards, modals
- **Full**: 9999px (rounded-full) - Pills, avatars

### Shadows
- **Small**: `0 1px 2px 0 rgb(0 0 0 / 0.05)` - Subtle elevation
- **Medium**: `0 4px 6px -1px rgb(0 0 0 / 0.1)` - Cards, dropdowns
- **Large**: `0 10px 15px -3px rgb(0 0 0 / 0.1)` - Modals, popovers
- **Hover**: Increase shadow on interactive elements

### Responsive Breakpoints
- **Mobile**: < 640px (sm)
- **Tablet**: 640px - 1024px (md, lg)
- **Desktop**: ≥ 1024px (xl, 2xl)

## Component Patterns

### Buttons

#### Primary Button
- **Background**: Primary Blue (#2563EB)
- **Text**: White, Semibold (600)
- **Padding**: 12px 24px (md), 16px 32px (lg)
- **Border radius**: 8px (rounded-lg)
- **Hover**: Darker blue (#1D4ED8), lift effect (translate-y-1)
- **Shadow**: Medium shadow, increases on hover

#### Secondary Button
- **Background**: Transparent
- **Border**: 2px solid Primary Blue
- **Text**: Primary Blue, Semibold (600)
- **Hover**: Background Primary Blue, text White

#### Disabled State
- **Background**: Gray 300
- **Text**: Gray 400
- **Cursor**: not-allowed
- **No hover effects**

### Cards
- **Background**: White
- **Border**: 1px solid Gray 200 (optional)
- **Border radius**: 8px (rounded-lg)
- **Padding**: 24px mobile, 32px desktop
- **Shadow**: Medium shadow
- **Hover**: Shadow increases, subtle scale (scale-105)

### Forms
- **Input height**: 48px (touch-friendly)
- **Padding**: 12px 16px
- **Border**: 1px solid Gray 300
- **Border radius**: 6px (rounded-md)
- **Focus**: Primary Blue ring, 2px offset
- **Error**: Red border, red text below
- **Labels**: Gray 700, Semibold (600), 14px

## Brand Applications

### Homepage Hero
- **Layout**: Centered text with book preview/animation below
- **Headline**: H1, Poppins Bold, "Erstelle magische Geschichten, in denen dein Kind der Held ist"
- **Subheadline**: H3, Inter Regular, "KI-gestützte personalisierte Kinderbücher in Minuten. Dein erstes Buch ist kostenlos."
- **CTAs**: 
  - Primary: "Jetzt kostenlos erstellen" (Magical Purple button)
  - Secondary: "Beispiele ansehen" (Outline button)
- **Trust elements**: "Sicher für Kinder" | "Hochwertige Druckqualität" | "Versand in 5-10 Tagen"
- **Background**: Soft gradient (white to pastel purple/blue)

### Book Type Cards
- **Layout**: 3-column grid desktop, 1-column mobile
- **Card structure**: 
  - Book illustration (16:9 aspect ratio)
  - Age badge ("2-4 Jahre")
  - Title (H3, "Bilderbuch")
  - Description (2-3 sentences)
  - Features (4-5 items with checkmarks)
  - Price ("€35 gedruckt")
  - CTA ("Jetzt erstellen")
- **Padding**: 24px
- **Border radius**: 16px (more rounded, child-friendly)
- **Hover**: Gentle lift, soft shadow increase, slight scale

### Sample Book Cards
- **Image**: 4:3 aspect ratio, book cover mockup
- **Overlay**: Soft gradient overlay on hover
- **Content**: 
  - Book title (H4, story name)
  - Age tag ("5-7 Jahre")
  - Preview badge ("Vorschau ansehen")
- **Hover effect**: Book "opens" slightly, CTA appears
- **CTA**: "Geschichte lesen" appears on hover

### Email Templates
- **Header**: Logo left (120px width), tagline right
- **Body**: Max width 600px, 16px body text
- **CTAs**: Full-width buttons on mobile, inline on desktop
- **Footer**: Gray 100 background, contact info, social links

### Social Media
- **Profile image**: Sparkee Studios icon, Primary Blue background
- **Cover images**: 1200x630px, brand colors, clear typography
- **Post graphics**: Consistent template with logo, primary colors
- **Aspect ratios**: 1:1 (Instagram), 16:9 (LinkedIn), 2:1 (Twitter)
